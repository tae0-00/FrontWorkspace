1. 속성 선택자

스타일을 부여하고자 하는 요소 내에 작성되어있는 속성을 이용해서 요소를 선택하는 방법 선택자 뒤에 [] 를 붙여 속성과 속성값을 제시한다. 

        선택자 [속성= 속성값]
        선택자[속성~=속성값]
        선택자[속성|=속성값]
        선택자[속성^=속성값]
        선택자[속성$=속성값]
        선택자[속성*=속성값]

2. 자손 선택자와 후손 선택자

자손 : 바로 하위 요소들 후손 : 하위요소 전체 

        *자손선택자(>)
        a>b{
            ...
        }

        *후손 선택자( )
        a b {
            ...
        }

3. 동위레벨 선택자

-동위관계에 있는 특정요소를 선택할떄 사용한다 (뒤에있는 요소만 가능) 

        *a 요소 뒤에있는 "하나만" 선택하기
        a+b {
            ...
        }
        * a 요소 뒤에 있는 모든 요소 건택
        a~b{
            ...
        }

4. 반응선택자

사용자의 움직임에 따라 선택이 되거나 안되는 선택자 

        *해당 요소에 클릭이 되었을 경우 스타일을 부여
        선탹자 : active{
            ...
        }
        * 해당 요소에 마우스가 올라가는 순간 스타일을 부여
        선택자 : hover{
            ...
        }

        선택자 : chacked{
            ...
