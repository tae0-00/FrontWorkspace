요소의 영역 관련 속성

        HTML의 요소는 총  4가지 영역으러 구성되어있다
        1. content 영역
         - 요소의 내용이 작성되는 영역
         - 시작태그오 종료태그 사이에 작성되는 내용

        2. padding 영역
         - content영역과 border영역 사이

        3. border 영역
         - 요소의 테두리 영역
         - content보다 바깥에서 content를 감싸고 있다.

        4. margin 영역
         - 현재요소와 다른 요소와의 간격을 나타내는 영역
