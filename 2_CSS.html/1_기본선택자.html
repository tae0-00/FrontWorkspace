CSS 선택자의 종류

1. 모든 선택자(*)

모든 요소들을 다 선택하고자 할때 사용한다 

            *{
                스타일속성 : 값;
                스타일 속성 : 값;
                ...
            }

2. 태그 선택자

해당 문서 내에 같은 이름의 태그들을 모두 선택할 때 사용 

            태그명{
                속성: 값; 
            }

3.클래스 선택자(.클래스명)

해당 문서 내에서 내가 원하는 "여러개의" 요소를 선택하고자 할 떄 사용 

            .클래스명{
                속성: 값;
            }

4. 아이디선택자 #아이디명

해당 문서 내에 특정 html 요소 "하나"만을 선택하고자 할 때 사용 단, 해당 요소의 id속성을 이용하여 고유한 아이디값을 부여해야한다. 

            #아이디명{
                ...
            }
