Javascript 개요

        -html의 조작과 변경을 담담하는 언어
        -hmtl로 웹문서의 구조를 만들고, css로 디자인을 정의하며, JS로 웹페이지상의 이벤트가 발생했을때 실핼 할 동ㄴ작을 정의한다.
        -웹 브라우저의 실행 가능한 인터프리터의방식의 스크립트 언어
        *인터프리터: 컨파일 없이 한줄 씩 실핼되는 언어

        장점 
         - 코드를 작성한 후 바로 실행결과를 볼수 있기 때문애 간단한 개발에 용이하다.
         -클래스, 접근제한자, 반환형 등 고급개념 없이 바로 사용가능 하기 때문에 초보자가 다루기 쉽다.
         -모든 ㅜ엡 브라우저의 표준 스크립트 언어이기ㅣ 때문에 범용성이 넓다.

         단점
          - 컴파일방식의 언어보다 실행속도가 느리다..
          - 큰 규모의 어플리케이션 개벌에 있어서 타입안전성이 부족하다..

1. 자바스크립트 사용방법

  1. inine: 이벤트를 부요하고자 하는 요소 내에 소스코드를 작성하는 방법
  2. internal : html문서 내에 script태그 영역에 자바스크립트를 작성하는 방법
  3. external: 소스 코드를 별도의 .js 파일로 작성하여 불러오는 방법
